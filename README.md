# Haikei Studio

Windows 版ダウンロード:

- [zip版をダウンロード](https://github.com/yamaguchi-yudai0422/Haikei-Studio/raw/main/HaikeiStudio-win64.zip)
- [exe単体をダウンロード](https://github.com/yamaguchi-yudai0422/Haikei-Studio/raw/main/Haikei%20Studio.exe)

## 使い方

1. `HaikeiStudio-win64.zip` をダウンロード
2. そのまま使いたい場合は `Haikei Studio.exe` 単体でもダウンロード可能
3. zip を解凍
4. `Haikei Studio.exe` を起動

## 画像について

- このアプリには最初からイラスト画像は入っていません
- 使いたい画像は自分で追加してください
- 追加した画像はアプリ内のライブラリに表示されます

## 画像の入れ方

1. アプリを一度起動する
2. アプリのあるフォルダに `library-assets` が作られる
3. `library-assets` の中に使いたい画像を入れる
4. 画像を入れたあと、アプリを開き直すかライブラリを更新する

対応形式:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.svg`

おすすめ:

- 背景透過PNG
- 背景画像は大きめサイズ
- 家具や小物は透過画像

## 補足

- 初回起動時に `library-assets` `projects` `exports` フォルダが自動作成されます
- 画像は `library-assets` に入れるとアプリへ追加されます
- PNG書き出しは `1920x1080` です
