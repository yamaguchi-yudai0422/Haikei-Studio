# Haikei Studio

Windows 版ダウンロード:

- [Haikei Studio v1.0.0 をダウンロード](https://github.com/yamaguchi-yudai0422/Haikei-Studio/releases/tag/v1.0.0)

## 使い方

1. Release ページを開く
2. `HaikeiStudio-win64.zip` をダウンロード
3. zip を解凍
4. `Haikei Studio.exe` を起動

## 補足

- 初回起動時に `library-assets` `projects` `exports` フォルダが自動作成されます
- 画像は `library-assets` に入れるとアプリへ追加されます
- PNG書き出しは `1920x1080` です
